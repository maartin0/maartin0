Hey 👋, I'm Martin, in my second year of Computer Science at the University of Southampton.

### Some projects I'm proud of
- [PoliSense](https://polisense.org) - A political news sentiment tracker using a RAG pipeline to generate daily questions based on recent global and local news
- BaTaK - A reaction time game using arcade buttons in a frame for [SUFST](https://github.com/sufst/pcb), with a custom PCB using an ESP32 and a dot matrix screen connected via an FPC
- [OrbY](https://maartin0.github.io/orby) - A 3D visualisation of the solar system using three.js, won Gold in the 2023 British Physics Olympiad (BPhO) computational physics challenge

#### Hackathon projects

- [SotonHack 2026](https://wtfismonad.online/) - Claude code for DJs, using facial recognition to determine crowd energy levels, placed top 3 of 32 teams ([devpost](https://devpost.com/software/wtf-is-monad))
